# AIDA: Классика в эпоху ИИ 🤖

Фреймворк AIDA (Attention, Interest, Desire, Action) остается фундаментом, но ИИ меняет способ его реализации.

## Структура
1. **Attention (Внимание)** — Использование ИИ для генерации провокационных заголовков или необычных визуальных метафор.
2. **Interest (Интерес)** — Персонализация контента на основе данных о сегментах аудитории.
3. **Desire (Желание)** — Демонстрация ценности через кейсы и сторителлинг, адаптированный под tone-of-voice бренда.
4. **Action (Действие)** — Тестирование различных вариантов CTA (Call to Action) для повышения конверсии.
